*Al Sharq Al Arabi Website*

Odai Al Ghanem

03/03/2023

INTRODUCTION PURPOSE

The Alsharq Al Arabi Sweet website is an e-commerce platform designed to sell a variety of sweets and desserts in Irbid, Jordan. The website provides an online platform for customers to browse and purchase products from the comfort of their homes.

INTENDED AUDIENCE AND PERTINENT SECTIONS

The website is intended for customers in Irbid, Jordan, who are looking for a convenient way to purchase sweets and desserts online. The website is divided into several sections, including home, about us, products, cart, and contact us.

PROJECT SCOPE

The project scope includes developing a user-friendly website with a responsive design that provides a seamless shopping experience for customers. The website should allow customers to browse and purchase products, view product details, add items to a cart, and checkout securely.

TECHNOLOGIES

The Alsharq Al Arabi Sweet website is built using the .NET MVC framework. The website is hosted on a Microsoft Azure server, and the front-end is built using HTML, CSS, and JavaScript. The website also utilizes a SQL Server database to store product information and customer data.

REFERENCES

The Alsharq Al Arabi Sweet website is designed based on the best practices for e-commerce websites, including user-centered design principles and secure payment processing. The website also adheres to the latest web development standards, including responsive design and cross-browser compatibility.

DESCRIPTION

The website features a simple and intuitive design that allows customers to easily navigate through the website and find the products they are looking for. The website displays a variety of sweets and desserts, including cakes, pastries, and Arabic sweets. Customers can view product details, including descriptions, images, and prices, and add items to their cart for purchase. The website also features a secure checkout process that ensures customer data is protected during transactions.

OPERATING ENVIRONMENT

The Alsharq Al Arabi Sweet website is designed to be compatible with modern web browsers, including Google Chrome, Mozilla Firefox, and Microsoft Edge. The website is also optimized for mobile devices, allowing customers to easily shop on their smartphones and tablets.

FEATURES

  • Easy navigation
  • Product categories
  • Product search
  • Product details page
  • Cart management
  • Secure checkout
  • Order history
  • Contact us form

DOCUMENTATION

The website includes detailed documentation on how to use the website, including instructions on how to browse products, add items to the cart, and checkout securely. The website also includes a FAQ section that answers common questions from customers.

FLOW CHART DIAGRAM

The website's flow chart diagram illustrates the process flow of the website, including the steps for browsing products, adding items to the cart, and completing the checkout process.

DATABASE SCHEMA

The website's database schema includes tables for storing product information, customer data, and order history. The database schema is designed to ensure data integrity and security.
